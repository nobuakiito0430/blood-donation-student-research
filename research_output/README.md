## 一番重要な結論

大学生の献血離れは、単純な「利他心の不足」では説明しにくい。文献上は、恐怖・不安、時間や機会のなさ、献血の流れが分からないこと、体重・貧血などの自己適格性への不安、学内献血機会の減少が大きい。一方、リピーター化には、初回体験の満足度、献血者としての自己効力感・アイデンティティ、友人からの誘い、具体的な必要性の可視化が効きやすい。

献血車施策としては、初回参加者向けには「友人と来る」「不安を減らす」「短時間で済む」「その場で意味が分かる」設計が重要。リピーター向けには「次回予約」「前回の貢献のフィードバック」「友人紹介」「大学内での小さなコミュニティ化」を測定可能な形で実施するのがよい。

## すぐ使える研究設計

学内献血車で、クラスターまたは日程単位で施策を分ける。

1. 通常告知
2. 友人同伴メッセージ
3. リアルタイム在庫・不足血液型メッセージ
4. 不安軽減メッセージと流れの可視化
5. 次回予約・紹介カード付き

主要アウトカムは、献血受付数、実献血数、初回率、友人同伴率、次回予約率、3か月・6か月以内の再来、紹介経由数。補助アウトカムは、恐怖、不安、主観的規範、自己効力感、献血者アイデンティティ、満足度。
