## ざっくり結論

大学生の献血離れは、単純な「利他心がない」では説明しにくいです。文献上は、恐怖・不安、時間や機会のなさ、献血の流れが分からないこと、体重・貧血などの自己適格性への不安、学内献血機会の減少が大きいと考えられます。

一方、リピーター化には、初回体験の満足度、自己効力感、献血者としてのアイデンティティ、友人からの誘い、具体的な必要性の可視化が重要です。

## 研究で使えそうな施策

- 友人同伴予約枠・紹介QR
- 献血の流れ、平均所要時間、痛み・副反応対応の掲示
- 今日必要な人数、不足血液型、地域在庫の可視化
- 献血後の次回予約、ラブラッド登録、通知許諾
- 献血不可だった学生向けの次回条件・健康情報・友人紹介導線
- 学生アンバサダー、サークル・ゼミ単位の小目標
- 献血後の貢献フィードバック

## 分析手法

- ロジスティック回帰
- Poisson / Negative Binomial回帰
- Difference-in-Differences
- Survival / Hazard model
- Mediation analysis / SEM
- Social Network Analysis
